# Dependencies
This plugin is depending on the [DokusCore](https://github.com/dokucore) Framework!<br>
Make sure that DokusCore is started before this plugin!

---
# Installation
Simply extract the `DokusCore--SkinCreator` into your `resources` folder, <br>
and start it via `ensure DokusCore--SkinCreator`.

---
# Plugin Information
This plugin your users the ability to set a custom skin for their <br>
characters. You can adjust things like the skin tone, wrinkles, lipstick,
hair, hair color, beard, beard color and many more.
